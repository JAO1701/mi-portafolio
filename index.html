// SOBRE MI

¡Hola a todos!. Soy Alejandro Oliveira, un programador y analista de datos autodidacta con una profunda pasión por la tecnología. Mi objetivo es claro: Transformar datos crudos en información valiosa y construir soluciones de software robustas y eficientes.

Cuento con experiencia en el ecosistema JavaScript (ES6+) para el desarrollo web; y utilizo SQL y Python para el análisis de datos e inteligencia artificial. Me especializo en el desarrollo backend, la automatización de procesos, y la gestión completa del ciclo de vida de los datos empresariales: ETL, preprocesamiento y visualización interactiva. En cuanto al desarrollo serverless y el ecosistema cloud, me he enfocado en AZURE, aunque también estoy familiarizado con AWS y GCP.

Me entusiasma aprender nuevas tecnologías. Siempre estoy dispuesto a trabajar en proyectos que requieran pensamiento crítico y analítico para desentrañar el verdadero potencial de los datos y la tecnología, creando soluciones innovadoras y escalables.

> ChatBot_Whatsaap_SQL

Un chatbot implementado con whatsaapweb.js, que se conecta con una base de datos privada (Sqlite-Cloud) y con un modelo de inteligencia artificial de OpenAi (gpt-4o-mini) para generar consultas SQL en el backend y responder al usuario mediante la interfaz de whatsaap.

Ver Repositorio en GitHub

> Data analysis-Historial Netflix

Este es un proyecto de análisis de datos, que mediante un script de Python, se encarga de procesar y analizar un archivo CSV con tu historial de visualizaciones de Netflix y generar un dashboard interactivo en HTML, implementando la librería Apexchart.js para la renderización.

> Reporte de Dashboard por Email

Esta aplicación automatiza el proceso de generar un reporte de ventas basado en datos de un archivo CSV, crea una captura de pantalla del reporte renderizado en HTML con playwright y la envía por correo electrónico mediante Mailjet.

Ver Repositorio en GitHub
